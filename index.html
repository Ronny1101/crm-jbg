Configuración inicial
Solo necesitas hacer esto una vez. Sigue los 3 pasos para conectar tu base de datos.
1Crea tu proyecto Firebase
Ve a console.firebase.google.com → Crear proyecto → ponle nombre (ej: crm-renting) → desactiva Analytics → Crear.

En el menú izquierdo: Firestore Database → Crear base de datos → modo prueba → Siguiente → Listo.
2Registra tu app web
En Firebase Console → ícono ⚙️ → Configuración del proyecto → sección "Tus apps" → clic en </> → ponle nombre → Registrar app.

Copia los valores del objeto firebaseConfig que aparece.
